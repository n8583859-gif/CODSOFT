# Simple Calculator (Python)

This is a simple command-line calculator built using Python.
It allows users to perform basic arithmetic operations with a clean and user-friendly interface.

## Features

* Supports addition, subtraction, multiplication, and division
* Handles invalid inputs safely
* Prevents division by zero
* Allows continuous calculations without restarting
* ask to user for quit or proceed

## How to Run

1. Run the Python file
2. Follow the on-screen instructions
3. Enter numbers and choose an operation

## Purpose

This project was built as part of my CodSoft internship to practice clean coding, input validation, and user-friendly program design.
