# Rock–Paper–Scissors Game (Python)

A clean and interactive command-line implementation of the classic Rock–Paper–Scissors game.
This project focuses on structured design, input validation, and a smooth user experience.

## Features

* Simple and intuitive gameplay
* Validated user input (prevents invalid entries)
* Random computer moves
* Accurate win/lose/tie logic
* Score tracking across multiple rounds
* Continuous play without restarting the program

## How It Works

1. The user selects Rock, Paper, or Scissors
2. The computer randomly selects its move
3. The winner is determined based on game rules
4. The result and updated score are displayed
5. The user can choose to play again or exit

## Project Structure

The code is designed with a modular approach:

* Input handling (user choice, replay control)
* Game logic (winner determination)
* Output display (results and score)
* Flow control (game loop)

## How to Run

1. Run the Python file
2. Follow the on-screen instructions
3. Play multiple rounds or exit anytime

## Purpose

This project was built as part of the CodSoft internship to practice:

* clean code structure
* separation of concerns
* input validation and error handling
* user-focused program design
