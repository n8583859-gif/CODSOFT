# Contact Book (Python)

A simple and structured command-line Contact Book application built using Python.
This project allows users to manage contacts efficiently with a clean interface and reliable input handling.

## Features

* Add new contacts (name, phone, email, address)
* View all saved contacts
* Search contacts by name or phone number
* Update existing contact details
* Delete contacts with confirmation
* Input validation for phone numbers and email
* Exit option

## How It Works

1. Choose an option from the menu
2. Perform actions like add, view, search, update, or delete
3. Data is managed in memory during runtime
4. The program runs continuously until the user choose exit option

## How to Run

1. Run the Python file
2. Follow the menu instructions
3. Manage contacts easily through options
